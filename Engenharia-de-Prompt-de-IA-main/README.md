# ℹ️ Atividades acadêmicas propostas em sala.

* **Ferramentas:** LLMs propostas em sala.

Aprendizados em avaliação de IA, desenvolvivemento web com IA e análise das mesmas.
* **Redução de Ruído:** Aprendi a aplicar técnicas de prompts em IA para atingir resultados esperados.
