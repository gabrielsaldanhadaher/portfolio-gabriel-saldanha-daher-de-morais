# Como Usar Videoconferência no IT Connect

## 📹 Guia Completo para Entrar e Iniciar Reuniões

### 1. Entrar em uma Reunião Ativa

**Passo a passo:**

1. **Acesse a comunidade**
   - Na aba "Explorar" ou "Minhas Comunidades"
   - Toque na comunidade desejada

2. **Clique em "Videoconferência"**
   - Se houver uma reunião ativa, você verá a opção "Entrar na Reunião"
   - Se não houver, poderá iniciar uma nova

3. **Autorize câmera e microfone**
   - O navegador/app pedirá permissão
   - Clique em "Permitir"

4. **Participe da reunião**
   - Use os controles para ativar/desativar câmera e microfone
   - Compartilhe sua tela se necessário
   - Use o chat integrado

### 2. Iniciar uma Reunião

**Passo a passo:**

1. **Acesse a comunidade**
   - Você deve ser membro da comunidade

2. **Clique em "Videoconferência"**
   - Se não houver reunião ativa, clique em "Iniciar Reunião"

3. **A reunião será criada**
   - Um nome único será gerado automaticamente
   - Você será redirecionado para o Jitsi Meet

4. **Compartilhe o link**
   - Outros membros podem entrar clicando em "Videoconferência"
   - Eles verão a reunião ativa e poderão participar

### 3. Controles de Reunião

| Controle | Função |
|----------|--------|
| 🎤 Microfone | Ativar/desativar áudio |
| 📹 Câmera | Ativar/desativar vídeo |
| 📺 Compartilhamento | Compartilhar tela do seu dispositivo |
| 💬 Chat | Enviar mensagens de texto |
| ⚙️ Configurações | Ajustar qualidade de vídeo, áudio, etc. |

### 4. Encerrar uma Reunião

**Apenas admins podem encerrar:**

1. **Acesse a tela de videoconferência**
2. **Clique em "Encerrar Reunião"**
3. **Confirme a ação**
4. **A reunião será encerrada para todos**

## 🌐 Acessando pelo Navegador (Web)

Se você estiver usando o IT Connect no navegador:

1. **Clique em "Videoconferência"**
2. **Uma nova aba será aberta** com o Jitsi Meet
3. **Autorize câmera e microfone**
4. **Participe normalmente**

## 📱 Acessando pelo Mobile

Se você estiver usando o IT Connect no celular:

1. **Clique em "Videoconferência"**
2. **A reunião será integrada ao app**
3. **Use os controles do app para gerenciar câmera e microfone**

## 🔐 Segurança e Privacidade

### Comunidades Privadas
- Apenas membros aprovados podem entrar
- Você precisa ser aprovado pelo admin antes de participar

### Comunidades Públicas
- Qualquer membro pode entrar
- Não requer aprovação

### Dados de Reunião
- Todas as reuniões são registradas no banco de dados
- Admins podem ver histórico de reuniões
- Você pode acessar relatórios de participação

## 🐛 Troubleshooting

### Problema: Não consigo entrar na reunião

**Soluções:**
- Verifique se você é membro da comunidade
- Verifique sua conexão com a internet
- Tente recarregar a página
- Limpe o cache do navegador

### Problema: Câmera/Microfone não funcionam

**Soluções:**
- Verifique as permissões do navegador
- Teste câmera/microfone em outro app
- Reinicie o navegador
- Verifique se o dispositivo não está em modo silencioso

### Problema: Áudio/vídeo com lag

**Soluções:**
- Reduza a qualidade de vídeo
- Feche outras abas/apps
- Aproxime-se do roteador WiFi
- Use conexão com fio se possível

### Problema: Não consigo compartilhar a tela

**Soluções:**
- Verifique se o navegador permite compartilhamento
- Tente usar um navegador diferente
- Reinicie o navegador
- Verifique as permissões do sistema operacional

## 💡 Dicas e Truques

### 1. Melhor Qualidade de Vídeo
- Use conexão com fio (Ethernet) em vez de WiFi
- Feche outros apps que usam internet
- Reduza o número de participantes se possível

### 2. Economia de Dados
- Desative o vídeo se não for necessário
- Use apenas áudio em conexões lentas
- Reduza a qualidade de vídeo

### 3. Participação Eficiente
- Mude o nome para algo descritivo
- Use o chat para não interromper
- Mude de câmera se tiver múltiplas
- Use fones de ouvido para melhor áudio

### 4. Compartilhamento de Tela
- Compartilhe apenas a janela necessária (não a tela inteira)
- Feche abas/apps desnecessários antes
- Teste o compartilhamento antes da reunião importante
